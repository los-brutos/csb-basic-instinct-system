# CSB - Basic Instinct System

## Informations

Ce module implémente le système de jeu des Imagitateurs du JDR : le **Basic Instinct System** sur **Custom System Builder**.

Vous pouvez retrouver les Imagitateurs sur leur chaine [Les imagitateurs du JDR - YouTube](https://www.youtube.com/channel/UCfugznJG9IYjJB9pWC_e0FA)

Merci à eux pour l'autorisation de la publication de ce module.

## Installation

### Standard

Dans la liste des Modules, clique sur le bouton Installer pour le module CSB - Basic Instinct System. Le module s'installera tout seul.

### Manuelle

Télécharge l'archive contenant le module. La décompresser dans le répertoire modules de Foundry. Redémarre Foundry pour que le nouveau module soit reconnu.

### Configuration

Crée un monde avec le système Custom System Builder. 

Active le module CSB - Basic Instinct System.

Importe les templates qui sont dans les compendiums :

- Clique droit sur un Compendium pour afficher le menu

- Clique sur *Importer tout le contenu*

- Dans la fenêtre qui s'affiche, coche la case *Garder l'ID des documents*

- Clique sur le bouton *Importer*

Les templates sont présents dans ton monde et tu peux commencer utiliser le module.

A noter que cette opération d'import sera à faire à chaque mise à jour du module pour bénéficier des mises à jour.

## Comment utiliser ce module ?

Le système **Custom System Builder** permet de créer des modèles (templates) d'Acteurs et d'Objets. A travers cette fonctionnalité vous trouverez dans ce module le nécessaire pour créer vos feuilles de PJ

Le Template ne doit pas être utilisé comme tel, il serve à générer de nouveau Acteurs (PJ ou PNJ)

### Fonctionnement du Basic Instinct System

Lis le journal dans le compendium du Module : tu auras  toutes les billes pour jouer avec ce système.
